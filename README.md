# file-counter-cli

A simple Node.js CLI tool to count the number of lines in a text file.

## Features
- Count the number of lines in a file
- Count the number of words in a file
- Simple command-line interface
- Built using Node.js and Commander

## Installation
Clone the repository:
```bash
git clone https://github.com/DweetParikh/file-counter-cli.git
cd file-counter-cli
npm install

node index.js words <file name>

node index.js lines <file name>
